# 🌸 Pink Pixels 
A soft pink aesthetic college planner built using HTML, CSS and JavaScript. 
## Features  
- 🏠 Homepage with goals
- ✅ To-do lists
- 📅 Editable timetable
- 💖 Wishlist
- 📚 Semester dashboard
- ⭐ Monthly study tracker
- 📝 Subject notes
- 💾 Local Storage support
